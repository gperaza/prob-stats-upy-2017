Installing Jupyter and R

The easier way to install both Jupyter and R with the interface between them configured automatically is to install first the Anaconda Python distribution. After you have Anaconda installed do the following:

Windows

Open the program called "anaconda prompt" and input the following command: 

conda install -c r r-essentials

Linux and Mac

Run the following command in the terminal 

conda install -c r r-essentials
